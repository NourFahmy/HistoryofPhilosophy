This folder contains the code that produced the findings in the data story found under /docs. Please run the notebooks in the following order:
Data preprocessing - the preprocessing needed to produce the findings in LDA. You can run this yourself, or use the dataset produced in /data.
EDA - the exploratory data analysis code & corresponding figures, findings and output that is included in the EDA section of my report.
LDA - the LDA code & corresponding figures, findings and output that is included in the LDA section of the report.
Please note that any substantial analysis of my findings will be found in my data story under /docs!
